source code 

Return device information for all present devices, filtering if requested by vendor and/or product IDs on USB devices.

randmac(length=8)

source code 

Returns a random MAC address using a list valid OUI's from ZigBee device manufacturers. Data is returned in air-format byte order (LSB first).

Parameters:
